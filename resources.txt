deploy en GitHub pages
1.- instalar en el proyecto: npm install gh-pages --save-dev
2.- configurar .json/script: "deploy": "gh-pages -d src"
3.- subir todos estos cambios al repository
4.- ejecutar en consola: npm run deploy
    ejecutarlo cada vez que haga actualizaciones en el sitio
